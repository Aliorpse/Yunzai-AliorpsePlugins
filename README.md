# Yunzai-AliorpsePlugins

依照个人Yunzai使用习惯所写的插件库

#### 安装与使用

- 安装: 下载js文件并移动到Yunzai/plugins/example目录下
- 用法: 详见各js插件内部注释

#### 未来可能会填补的功能
如果你对这些功能有需要，请发个issue来催更

- bilitv.js - 解析直播链接，个人资料链接
